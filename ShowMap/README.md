# Show Map
Displays a map using Google or Bing

* `show-map brisbane`
* `show-map helsinki`
* `show-map 10017`
* `show-map 10017 -UseBing`

Also, if you copy a location to the clipboard and run `show-map`, it will take what the text from the clipboard and display that
